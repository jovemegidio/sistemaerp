Requisitos do Sistema

Antes de iniciar a instalao, verifique se seu computaçãor atende aos requisitos mnimos:

Sistema Operacional

Windows 10 ou superior (64-bit)

Memria RAM

Mnimo 4GB (recomendação 8GB)

Espao em Disco

Mnimo 2GB livres

Navegaçãor

Chrome, Edge ou Firefox (atualizaçãos)

Softwares Necessrios

O Aluforce ERP requer os seguintes softwares instalaçãos:

Node.js (v18 ou superior)

Motor JavaScript do sistema

Baixar Node.js
MySQL Server 8.0

Banco de daçãos do sistema

Baixar MySQL
Importante - Instalao do MySQL

Durante a instalao do MySQL, configure a senha do usurio root como: @dminalu

Se voc usar uma senha diferente, ser necessrio alterar no arquivo .env aps a instalao.

Passo a Passo da Instalao

1

Extraia os arquivos do sistema

Extraia a pasta "Sistema - Aluforce v.2 - BETA" para um local de fcil acesso, como C:\Aluforce ou C:\Sistemas\Aluforce.

2

Execute o instalaçãor como Administraçãor

Localize o arquivo INSTALAR.bat na pasta do sistema.

Clique com o boto direito e selecione "Executar como administraçãor".

3

Aguarde a verificao automtica

O instalaçãor ir verificar automaticamente:

  • Node.js instalação
  • MySQL Server instalação
  • Conexo com banco de daçãos
  • Configuraes do sistema
4

Clique em "Iniciar Instalao"

Aps a verificao, clique no boto verde para iniciar o processo de instalao.

O instalaçãor ir:

  • Criar o banco de daçãos (se necessrio)
  • Importar a estrutura e daçãos iniciais
  • Instalar todas as dependncias
  • Criar atalhos na rea de trabalho
5

Inicie o sistema

Aps a instalao, clique em "Sim" quando perguntação se deseja iniciar o sistema.

Ou use o atalho "Iniciar Aluforce Server" criação na rea de trabalho.

Pronto!

Aps iniciar o servidor, acesse o sistema pelo navegaçãor em: http://localhost:3000

Iniciando o Sistema no Dia a Dia

Para iniciar o Aluforce ERP aps a instalao:

1

Inicie o Servidor

D duplo clique no atalho "Iniciar Aluforce Server" na rea de trabalho.

Uma janela do terminal ser aberta mostrando que o servidor est rodando.

2

Acesse o Sistema

D duplo clique no atalho "Aluforce ERP" na rea de trabalho.

Ou abra o navegaçãor e acesse: http://localhost:3000

Dica

Mantenha a janela do servidor (terminal preto) sempre aberta enquanto estiver usando o sistema. Fechar essa janela ir encerrar o Aluforce.

Perguntas Frequentes

O sistema no abre no navegaçãor
Verifique se o servidor est rodando. A janela do terminal deve mostrar "Servidor rodando na porta 3000".
Erro de conexo com banco de daçãos
Verifique se o servio do MySQL est rodando. Abra "Servios" do Windows (services.msc) e procure por "MySQL80" ou similar.
Porta 3000 em uso
Outro programa est usando a porta. Reinicie o computaçãor ou altere a porta no arquivo .env (varivel PORT).
Como atualizar o sistema
Substitua os arquivos do sistema pelos novos, mantendo o arquivo .env. O banco de daçãos ser preservação automaticamente.
Como fazer backup do banco
Use o MySQL Workbench ou o comando: mysqldump -u root -p aluforce_vendas > backup.sql
