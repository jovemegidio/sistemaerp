Exportar Relatrios para Excel

Atualização h 1 semana 5 min de leitura

Todos os relatrios do Aluforce podem ser exportaçãos para Excel, permitindo anlises personalizadas e compartilhamento com sua equipe.

Exportando um relatrio

  1. 1
    Acesse o relatrio desejação

    Navegue at o mdulo e selecione o relatrio.

  2. 2
    Aplique os filtros

    Defina perodo, categorias e outros parmetros.

  3. 3
    Gere o relatrio

    Clique em "Gerar" ou "Consultar".

  4. 4
    Clique em "Exportar"

    Selecione o formato Excel (.xlsx).

Formatos disponveis

Formato Extenso Uso recomendação
Excel .xlsx Anlises, grficos, manipulao de dados
CSV .csv Importao em outros sistemas
PDF .pdf Apresentaes, arquivamento

Dicas para melhor aproveitamento

Tabelas Dinmicas

Use os dados exportaçãos para criar Tabelas Dinmicas no Excel e fazer anlises cruzadas.

Boas prticas

  • Exporte apenas os dados necessrios para evitar arquivos muito grandes
  • Use filtros no sistema antes de exportar
  • Organize os arquivos por data e tipo de relatrio
  • Faa backup regular dos relatrios importantes

Limitaes

  • Mximo de 50.000 linhas por exportao
  • Alguns campos calculaçãos podem precisar de ajuste
  • Formatao visual do sistema no mantida
