Entrada de NF-e

Atualizado há 1 semana • 8 min de leitura

A entrada de NF-e permite importar as notas fiscais recebidas de fornecedores, atualizando automaticamente o estoque e gerando as contas a pagar.

Formas de entrada de NF-e

Método Descrição Quando usar
Importação XML Upload do arquivo XML da NF-e Quando tem o arquivo XML
Consulta SEFAZ Busca automática de notas emitidas contra seu CNPJ Notas ainda não recebidas
Digitação manual Entrada manual dos dados da nota Quando não tem XML disponível
Chave de acesso Busca pela chave de 44 dígitos Quando tem apenas a chave

Importação por XML

  1. 1
    Acesse Compras > Entrada de NF-e

    Navegue até o módulo de entrada de notas.

  2. 2
    Clique em "Importar XML"

    Selecione a opção de importação.

  3. 3
    Selecione o arquivo

    Escolha o XML da NF-e ou arraste para a área indicada.

  4. 4
    Revise os dados

    O sistema exibirá os dados da nota para conferência.

  5. 5
    Vincule os produtos

    Associe os itens da nota aos seus produtos cadastrados.

Vinculação de produtos

Ao importar uma NF-e, é necessário vincular os produtos do fornecedor aos seus produtos:

Vinculação automática

Se você já cadastrou o código do fornecedor no produto, a vinculação será automática nas próximas importações.

Vinculação manual

  1. 1
    Identifique o item

    Verifique a descrição do produto na NF-e.

  2. 2
    Busque seu produto

    Pesquise pelo código ou descrição no seu cadastro.

  3. 3
    Confirme a vinculação

    Salve para usar nas próximas importações.

Consulta na SEFAZ

O Aluforce pode buscar automaticamente as NF-e emitidas contra o seu CNPJ:

  1. 1
    Acesse "Consulta SEFAZ"

    Na tela de entrada de NF-e, clique na opção.

  2. 2
    Informe o período

    Selecione as datas para busca.

  3. 3
    Aguarde a consulta

    O sistema buscará as notas disponíveis.

  4. 4
    Selecione as notas

    Escolha quais notas deseja importar.

O que acontece na entrada

Módulo Ação
Estoque Entrada automática dos produtos
Financeiro Criação de contas a pagar
Fiscal Registro para escrituração
Custo Atualização do custo dos produtos

Importante

Confira sempre se a unidade de medida e a quantidade estão corretas antes de confirmar a entrada.
