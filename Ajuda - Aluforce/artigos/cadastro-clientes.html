Cadastro de Clientes

Atualizado há 1 semana • 8 min de leitura

O cadastro de clientes é fundamental para o funcionamento do seu sistema Aluforce. Neste artigo, você aprenderá como cadastrar, editar e gerenciar seus clientes de forma eficiente.

Acessando o cadastro de clientes

Para acessar o módulo de cadastro de clientes:

  1. 1
    Acesse o menu Cadastros

    No menu lateral, clique em "Cadastros" para expandir as opções.

  2. 2
    Selecione "Clientes"

    Clique na opção "Clientes" para abrir a lista de clientes cadastrados.

  3. 3
    Clique em "Novo Cliente"

    Use o botão "Novo" ou "+" para iniciar um novo cadastro.

Informações obrigatórias

Para cadastrar um cliente, você precisará preencher os seguintes campos obrigatórios:

Campo Descrição Formato
Razão Social / Nome Nome completo ou razão social do cliente Texto livre
CPF / CNPJ Documento de identificação 000.000.000-00 ou 00.000.000/0000-00
Inscrição Estadual IE do cliente (se houver) Numérico
Endereço Endereço completo para faturamento Texto livre
CEP Código postal 00000-000
Cidade / UF Município e estado Seleção

Informações complementares

Além dos dados obrigatórios, você pode adicionar:

  • Contatos: Telefone, celular, e-mail e site
  • Endereço de entrega: Se diferente do endereço principal
  • Dados bancários: Para recebimentos ou pagamentos
  • Observações: Informações adicionais relevantes
  • Tags: Categorização personalizada

Dica

Ao digitar o CNPJ, o sistema pode buscar automaticamente os dados cadastrais da empresa na Receita Federal. Essa funcionalidade agiliza o preenchimento do formulário.

Configurações de vendas

Na aba de configurações de vendas, você pode definir:

  • Tabela de preços: Qual tabela de preços será aplicada para este cliente
  • Condição de pagamento: Prazo padrão para pagamento
  • Limite de crédito: Valor máximo para vendas a prazo
  • Vendedor responsável: Vendedor vinculado ao cliente
  • Desconto padrão: Percentual de desconto automático

Importação em lote

Para importar múltiplos clientes de uma vez:

  1. 1
    Baixe o modelo de planilha

    Acesse Cadastros > Clientes > Importar e baixe o modelo Excel.

  2. 2
    Preencha os dados

    Complete a planilha com os dados dos clientes seguindo o formato indicado.

  3. 3
    Faça o upload

    Envie o arquivo e aguarde a validação dos dados.

  4. 4
    Confirme a importação

    Revise os dados e confirme para finalizar a importação.

Atenção

Verifique se não há duplicidade de CPF/CNPJ antes de importar. O sistema não permite cadastros duplicados.
