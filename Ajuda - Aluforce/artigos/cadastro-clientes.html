Cadastro de Clientes

Atualização h 1 semana 8 min de leitura

O cadastro de clientes fundamental para o funcionamento do seu sistema Aluforce. Neste artigo, voc aprender como cadastrar, editar e gerenciar seus clientes de forma eficiente.

Acessando o cadastro de clientes

Para acessar o mdulo de cadastro de clientes:

  1. 1
    Acesse o menu Cadastros

    No menu lateral, clique em "Cadastros" para expandir as opes.

  2. 2
    Selecione "Clientes"

    Clique na opo "Clientes" para abrir a lista de clientes cadastraçãos.

  3. 3
    Clique em "Novo Cliente"

    Use o boto "Novo" ou "+" para iniciar um novo cadastro.

Informaes obrigatrias

Para cadastrar um cliente, voc precisar preencher os seguintes campos obrigatrios:

Campo Descrio Formato
Razo Social / Nome Nome completo ou razo social do cliente Texto livre
CPF / CNPJ Documento de identificao 000.000.000-00 ou 00.000.000/0000-00
Inscrio Estadual IE do cliente (se houver) Numrico
Endereo Endereo completo para faturamento Texto livre
CEP Cdigo postal 00000-000
Cidade / UF Municpio e estação Seleo

Informaes complementares

Alm dos dados obrigatrios, voc pode adicionar:

  • Contatos: Telefone, celular, e-mail e site
  • Endereo de entrega: Se diferente do endereo principal
  • Daçãos bancrios: Para recebimentos ou pagamentos
  • Observaes: Informaes adicionais relevantes
  • Tags: Categorizao personalizada

Dica

Ao digitar o CNPJ, o sistema pode buscar automaticamente os dados cadastrais da empresa na Receita Federal. Essa funcionalidade agiliza o preenchimento do formulrio.

Configuraes de vendas

Na aba de configuraes de vendas, voc pode definir:

  • Tabela de preos: Qual tabela de preos ser aplicada para este cliente
  • Condio de pagamento: Prazo padro para pagamento
  • Limite de crdito: Valor mximo para vendas a prazo
  • Vendedor responsvel: Vendedor vinculação ao cliente
  • Desconto padro: Percentual de desconto automtico

Importao em lote

Para importar mltiplos clientes de uma vez:

  1. 1
    Baixe o modelo de planilha

    Acesse Cadastros > Clientes > Importar e baixe o modelo Excel.

  2. 2
    Preencha os dados

    Complete a planilha com os dados dos clientes seguindo o formato indicação.

  3. 3
    Faa o upload

    Envie o arquivo e aguarde a validao dos dados.

  4. 4
    Confirme a importao

    Revise os dados e confirme para finalizar a importao.

Ateno

Verifique se no h duplicidade de CPF/CNPJ antes de importar. O sistema no permite cadastros duplicaçãos.
