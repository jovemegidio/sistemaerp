Configurando uma integrao

  1. 1
    Acesse Configuraes > Integraes

    Veja todas as integraes disponveis.

  2. 2
    Selecione a integrao

    Clique na integrao desejada.

  3. 3
    Configure as credenciais

    Informe os dados de acesso fornecidos pelo parceiro.

  4. 4
    Teste a conexo

    Valide se a integrao est funcionando.

  5. 5
    Ative a integrao

    Habilite para uso em produo.

API REST

Para integraes personalizadas, disponibilizamos uma API REST completa.

Autenticao

Todas as requisies devem incluir o token de API no header:

Header de autenticao

Authorization: Bearer seu_token_aqui

Endpoints principais

Recurso Endpoint Mtodos
Clientes /api/v1/clientes GET, POST, PUT, DELETE
Produtos /api/v1/produtos GET, POST, PUT, DELETE
Pedidos /api/v1/pedidos GET, POST, PUT
Notas Fiscais /api/v1/nfe GET, POST
Financeiro /api/v1/financeiro GET, POST

Gerando token de API

  1. 1
    Acesse Configuraes > API

    Entre na rea de configurao da API.

  2. 2
    Clique em "Gerar Token"

    Um novo token ser criado.

  3. 3
    Copie e guarde o token

    O token exibido apenas uma vez.

Segurana

Nunca compartilhe seu token de API. Se suspeitar de vazamento, revogue imediatamente e gere um novo.
