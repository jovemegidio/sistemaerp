Integrações e API

Atualizado há 1 semana • 10 min de leitura

O Aluforce oferece diversas integrações nativas e uma API completa para conectar com outros sistemas da sua empresa.

Categoria Integrações

Configurando uma integração

  1. 1
    Acesse Configurações > Integrações

    Veja todas as integrações disponíveis.

  2. 2
    Selecione a integração

    Clique na integração desejada.

  3. 3
    Configure as credenciais

    Informe os dados de acesso fornecidos pelo parceiro.

  4. 4
    Teste a conexão

    Valide se a integração está funcionando.

  5. 5
    Ative a integração

    Habilite para uso em produção.

API REST

Para integrações personalizadas, disponibilizamos uma API REST completa.

Autenticação

Todas as requisições devem incluir o token de API no header:

Header de autenticação

Authorization: Bearer seu_token_aqui

Endpoints principais

Recurso Endpoint Métodos
Clientes /api/v1/clientes GET, POST, PUT, DELETE
Produtos /api/v1/produtos GET, POST, PUT, DELETE
Pedidos /api/v1/pedidos GET, POST, PUT
Notas Fiscais /api/v1/nfe GET, POST
Financeiro /api/v1/financeiro GET, POST

Gerando token de API

  1. 1
    Acesse Configurações > API

    Entre na área de configuração da API.

  2. 2
    Clique em "Gerar Token"

    Um novo token será criado.

  3. 3
    Copie e guarde o token

    O token é exibido apenas uma vez.

Segurança

Nunca compartilhe seu token de API. Se suspeitar de vazamento, revogue imediatamente e gere um novo.

Webhooks

Configure webhooks para receber notificações em tempo real de eventos no sistema:

  • NF-e autorizada
  • Pagamento recebido
  • Status de pedido alterado
