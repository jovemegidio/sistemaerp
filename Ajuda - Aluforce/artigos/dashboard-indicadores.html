Personalizando o Dashboard

Voc pode personalizar quais indicaçãores aparecem no seu Dashboard:

  1. Clique no cone de engrenagem no canto superior direito do Dashboard
  2. Selecione Personalizar Dashboard
  3. Arraste e solte os widgets para reorganizar
  4. Marque/desmarque indicaçãores para mostrar/ocultar
  5. Clique em Salvar para aplicar as alteraes
