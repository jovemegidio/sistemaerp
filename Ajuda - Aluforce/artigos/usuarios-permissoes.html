Usuários e Permissões

Atualizado há 1 semana • 10 min de leitura

O gerenciamento de usuários e permissões garante que cada colaborador tenha acesso apenas às funcionalidades necessárias para seu trabalho, aumentando a segurança do sistema.

Acessando o módulo

  1. 1
    Menu Configurações

    Clique no ícone de configurações.

  2. 2
    Usuários

    Acesse a gestão de usuários.

Cadastrando um novo usuário

  1. 1
    Clique em "Novo Usuário"

    O formulário de cadastro será exibido.

  2. 2
    Preencha os dados

    Nome, e-mail, telefone e departamento.

  3. 3
    Defina o perfil de acesso

    Selecione um perfil pré-definido ou personalize.

  4. 4
    Envie o convite

    O usuário receberá um e-mail para criar sua senha.

Perfis de acesso padrão

Perfil Descrição Acesso
Administrador Acesso total ao sistema Todas as funcionalidades
Financeiro Gestão financeira Contas, fluxo de caixa, relatórios
Vendedor Operações de venda Pedidos, clientes, consultas
Estoquista Controle de estoque Produtos, movimentações, inventário
Comprador Operações de compra Pedidos de compra, fornecedores
Visualização Apenas consulta Relatórios e consultas

Personalizando permissões

Para criar um perfil personalizado:

  1. 1
    Acesse Perfis de Acesso

    Em Configurações > Perfis.

  2. 2
    Crie um novo perfil

    Clique em "Novo Perfil".

  3. 3
    Marque as permissões

    Selecione cada funcionalidade permitida.

Tipos de permissão

Tipo Descrição
Visualizar Apenas consultar dados
Criar Adicionar novos registros
Editar Modificar registros existentes
Excluir Remover registros
Aprovar Autorizar operações

Boas práticas

Siga o princípio do menor privilégio: dê apenas as permissões necessárias para cada função.

Gerenciando usuários

Desativar usuário

Para colaboradores que saíram da empresa, desative o acesso em vez de excluir. Isso mantém o histórico de operações.

Resetar senha

Se um usuário esqueceu a senha, você pode enviar um link de redefinição pelo sistema.

Alterar perfil

Quando um colaborador muda de função, atualize seu perfil de acesso.

Atenção

Mantenha sempre pelo menos um usuário administrador ativo para evitar bloqueio de acesso.
