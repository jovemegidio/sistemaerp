Primeiros Passos Aluforce: 2. Configurações Iniciais

Atualizado há 1 mês • 8 min de leitura

Após o primeiro acesso, é hora de configurar o Aluforce de acordo com as necessidades da sua empresa. Este artigo mostra as configurações essenciais para começar a usar o sistema.

Dados da empresa

O primeiro passo é completar os dados cadastrais da sua empresa:

  1. Acesse Configurações > Empresa
  2. Preencha os dados do CNPJ, Razão Social e Nome Fantasia
  3. Adicione o endereço completo da empresa
  4. Configure os dados de contato (telefone, e-mail)
  5. Faça upload do logotipo da empresa

Configurações fiscais

Para emitir notas fiscais, configure:

  • Regime tributário: Simples Nacional, Lucro Presumido ou Lucro Real
  • Inscrição Estadual: Obrigatória para empresas que comercializam produtos
  • Inscrição Municipal: Necessária para prestadores de serviços
  • Certificado Digital: A1 ou A3 para assinatura de documentos fiscais

Configurando usuários

Adicione os colaboradores que utilizarão o sistema:

  1. Acesse Configurações > Usuários
  2. Clique em Novo Usuário
  3. Preencha nome, e-mail e defina uma senha temporária
  4. Selecione o perfil de acesso adequado
  5. O usuário receberá um e-mail para ativar a conta

Perfis de acesso disponíveis

  • Administrador: Acesso total ao sistema
  • Financeiro: Acesso aos módulos de finanças e relatórios
  • Vendedor: Acesso ao módulo de vendas e cadastros
  • Estoquista: Acesso ao módulo de estoque
  • Visualizador: Apenas visualização de dados

Configurações de notificações

Defina como você deseja receber alertas do sistema:

  • Notificações por e-mail
  • Alertas no sistema
  • Notificações push (no app mobile)
