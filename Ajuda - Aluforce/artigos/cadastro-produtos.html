Cadastro de Produtos

Atualizado há 1 semana • 10 min de leitura

O cadastro de produtos é essencial para vendas, compras e controle de estoque. Aprenda a cadastrar seus produtos de forma completa, incluindo informações fiscais.

Tipos de cadastro

O Aluforce permite cadastrar:

  • Produtos: Itens físicos que movimentam estoque
  • Serviços: Serviços prestados que não afetam estoque
  • Kits/Combos: Agrupamento de produtos vendidos juntos

Criando um novo produto

  1. 1
    Acesse Cadastros > Produtos

    No menu lateral, navegue até o cadastro de produtos.

  2. 2
    Clique em "Novo Produto"

    O formulário de cadastro será exibido.

  3. 3
    Preencha os dados básicos

    Informe descrição, código, unidade de medida e preços.

Dados básicos do produto

Informações fiscais

Configure corretamente os dados fiscais para emissão de notas:

Importante

Os dados fiscais são essenciais para a correta emissão de NF-e. Consulte seu contador para definir NCM, CFOP e tributação adequados.

Campos fiscais

  • NCM: Código de 8 dígitos que classifica o produto
  • CEST: Código Especificador da Substituição Tributária
  • Origem: Nacional, importado, etc.
  • CFOP padrão: Código fiscal da operação
  • ICMS: Configuração de ICMS por estado
  • PIS/COFINS: Tributação federal
  • IPI: Imposto sobre Produtos Industrializados

Controle de estoque

Na aba de estoque, configure:

  • Controla estoque: Sim/Não
  • Estoque mínimo: Quantidade para alerta de reposição
  • Estoque máximo: Limite de armazenamento
  • Local de estoque: Depósito ou localização
  • Lote/Validade: Se o produto exige controle de lote

Múltiplas unidades de medida

Configure conversão entre unidades:

Unidade Base Unidade Secundária Fator de Conversão
Metro (MT) Rolo (RL) 1 RL = 100 MT
Unidade (UN) Caixa (CX) 1 CX = 12 UN
Quilograma (KG) Tonelada (TON) 1 TON = 1000 KG

Importação de produtos

  1. 1
    Baixe a planilha modelo

    Em Produtos > Importar, baixe o template.

  2. 2
    Preencha os dados

    Complete todos os campos obrigatórios da planilha.

  3. 3
    Valide e importe

    Envie o arquivo e revise os dados antes de confirmar.

Atenção

Produtos com código duplicado serão atualizados. Certifique-se de usar códigos únicos para novos produtos.
