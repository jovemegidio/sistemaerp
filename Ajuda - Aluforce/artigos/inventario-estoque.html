Inventário de Estoque

Atualizado há 1 semana • 10 min de leitura

O inventário é o processo de contagem física dos produtos em estoque, comparando com os saldos registrados no sistema para identificar e corrigir divergências.

Quando fazer inventário?

  • Inventário periódico: Mensal, trimestral ou anual
  • Inventário rotativo: Grupos de produtos por vez
  • Inventário eventual: Após identificar problemas
  • Inventário fiscal: Para fechamento contábil

Criando um inventário

  1. 1
    Acesse Estoque > Inventário

    Navegue até o módulo de inventário.

  2. 2
    Clique em "Novo Inventário"

    Inicie um novo processo de contagem.

  3. 3
    Selecione os produtos

    Escolha todos ou filtre por categoria/local.

  4. 4
    Gere a lista de contagem

    O sistema criará a lista para impressão ou uso digital.

Realizando a contagem

Dica importante

Recomendamos fazer a contagem com o estoque "congelado", evitando movimentações durante o processo.

Métodos de contagem

Método Descrição
Lista impressa Imprima a lista e anote as quantidades manualmente
Coletor de dados Use um coletor para leitura de códigos de barras
Aplicativo móvel Conte usando smartphone ou tablet
Digitação direta Lance as quantidades diretamente no sistema

Lançando a contagem

  1. 1
    Acesse o inventário aberto

    Localize o inventário em andamento.

  2. 2
    Informe as quantidades

    Digite a quantidade contada para cada produto.

  3. 3
    Revise as divergências

    O sistema destacará diferenças entre contado e sistema.

Analisando divergências

Situação Possível causa
Físico maior que sistema Entrada não registrada, erro de contagem anterior
Físico menor que sistema Saída não registrada, perda, furto, erro de contagem

Finalizando o inventário

  1. 1
    Revise todas as divergências

    Certifique-se de que as diferenças estão corretas.

  2. 2
    Aprove o inventário

    Usuário com permissão deve aprovar os ajustes.

  3. 3
    Confirme os ajustes

    O sistema gerará as movimentações de ajuste automaticamente.

Atenção

Após finalizado, o inventário não pode ser alterado. Revise cuidadosamente antes de confirmar.
